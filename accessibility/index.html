Accessibility

Visually ambitious does not mean usability gets ignored.

ASH-TRA aims to build experiences that remain readable, structured, mobile-friendly, and usable even when the visual direction is bold and design-led.

Current accessibility priorities

  • Clear heading structure
  • Keyboard-friendly navigation
  • Visible focus states
  • Readable text contrast
  • Meaningful alt text
  • Reduced-motion support
  • Responsive tap targets

Ongoing improvement

If you encounter an accessibility issue, please report it through the contact form so it can be reviewed and improved.
