# ash-tra.com Web App

This folder contains the production-facing Next.js app for the ash-tra.com marketing site and client dashboard.

The root [`README.md`](/home/ash/nacho/README.md) is the canonical operator guide. Start there for:

- setup
- route overview
- environment variables
- storage behavior
- brand/media workflow

Useful local commands from this directory:

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
npm run media:generate
```
