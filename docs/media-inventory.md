# ASH-TRA Media Inventory

The new visuals are stored locally under `assets/media/` and use descriptive, SEO-aware filenames.

Hero assets cover every page except `discovery`, which keeps its current hero image. The hero filenames reflect page purpose and visual concept, for example `home-hero-orbital-launch-business-growth.svg` and `contact-hero-beacon-route-project-enquiry.svg`.

Support assets cover the homepage premise, process summary, contact routes, discovery support, examples support, and legal/system atlas sections. Overlay assets include route arrows and directional cues used across cards and support sections.
